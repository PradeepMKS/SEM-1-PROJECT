# Passed-out Student Tracker Backend

## Setup

1. `cd "d:\SEM 1 PROJECT\back end"`
2. `npm install`
3. `npm run start` (or `npm run dev` with nodemon)

## API
- GET /api/students
- GET /api/students/:id
- POST /api/students
- PUT /api/students/:id
- DELETE /api/students/:id

## Notes
- SQLite DB stored as `alumni.db` in backend folder
