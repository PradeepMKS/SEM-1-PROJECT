# Passed-out Student Tracker Frontend

## Run

1. Serve this folder on a local development server. Example:
   - `npx http-server . -p 8000` (install with `npm i -g http-server`)
   - Or using VS Code Live Server extension
2. Open http://127.0.0.1:8000
3. Ensure backend is running at http://localhost:5000

## Notes
- CRUD operations connect to backend API.
- Update API URL in `app.js` if backend port changes.
